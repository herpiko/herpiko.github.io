saya menulisnya di sini agar saya bisa mengingat kembali kebencian ini.

saya tidak akan menjadi orang lain jika anda bisa mengontrol diri dari paranoid-tidak-masuk-akal anda. saya juga sudah bersumpah tidak mempercayai siapapun tentang masa lalu termasuk anda--itu bukan urusan saya karena semuanya mutlak kesalahan tolol kalian jadi tidak ada gunanya anda mengungkit-ungkit masa lalu. anda seperti, saya sudah memikirkannya dalam waktu lama dan masih belum bisa menemukan kata yang tepat. dan siapapun pantas menertawakannya.

anda bisa menggunakan posisi anda untuk memojokkan saya, membangga-banggakan siapa anda dan meneteskan air mata setiap saya akan membuka mulut. tapi itu tidak mempan untuk saya karena saya bukan dia, dia atau dia. saya adalah saya. anda bisa melancarkan kutukan. anda bisa merubah reputasi saya dalam sekejap. anda bisa menghancurkan saya. anda bisa menjadikan saya sampah. silakan. saya sudah lelah tertawa. saya tidak akan melihat ke belakang karena waktu sudah terlalu lama berbohong.

anda akan berteriak kasar, saya tidak akan mendengarkan anda!

saya kehilangan kontak dengan beberapa orang bahkan dengan orang yang setiap harinya hanya berjarak tiga sampai sepuluh meter dari saya. ditambah anda lagi, tapi rasanya kali ini sama sekali tidak menyedihkan, persis seperti saat anda pertama kali memecahkan piring. terlalu mudah untuk mengelola rasa benci selevel ini. silakan menangis atas emosi kosong saya untuk hubungan ini.

anda sering bertanya kasar, "kamu tahu rasanya? kamu tahu rasanya? kamu tahu rasanya?". tentu saja saya tahu dan saya tidak perlu menunggu anda menjawab sendiri pertanyaan anda. saya membiarkan anda hidup tenang bertahun-tahun. saya tutup mulut dan membusuk hampir mati. heh, anda lihat dan dengar semuanya tapi berpura-pura bahwa saya sudah tidak hidup lagi.

bagaimana kalau saya menanyakan pertanyaan yang sama? saya tidak akan menanyakannya karena saya terlalu kasihan. saya takut anda tidak kuat menghadapinya, seperti yang saya lakukan pada yang lain. saya punya argumentasi baik dan akan semakin banyak sampai anda merasa putus asa untuk hidup. anda terlalu lemah!

bagaimana kita bisa tuntaskan masalah ini jika anda selalu merengek setiap saya berkata-kata?

saya tidak bilang saya benar sepenuhnya. saya ingin anda melihat akar semua masalah tolol ini. saya tidak peduli kesombongan anda, tidak mau mengakuinya, yang lain sudah meminta maaf atas kebodohan kalian dan sekarang saya menuntut permohonan maaf dari anda. saya tidak akan menuntut hak-hak saya, lupakan itu. saya hanya ingin anda minta maaf.
