terlambat dan saya panik karena sudah login dengan gmail. sampai keringat saya bercucuran dan jengkel sekali sambil sesekali melirik ke meja operator. keji benar. :(

nah, apa yang saya lakukan?

komputer tidak dideepfreeze maupun shadow. jelas, saya merusak windowsnya. saya menghapus ntldr, semua file di bawah system32 selain folder yang mungkin terhapus. membuat file .bat shutdown -r -t 00 dan memasang di startup all user via registry. mudahan saja diinstall ulang, amin. terakhir, mereset kasar PC-nya kemudian membayar billing dan tidak pernah muncul lagi di warnet itu.

semoga setimpal. :(
