Waktu itu hari senin, hati saya masih mengambang-ambang karena baru pulang dari Bali memenangkan olimpiade teknik 2008 pada mata lomba web design. Pagi itu tidak upacara karena sekolah hanya mengadakan upacara 2 kali sebulan, selang-seling.  Saya berjalan masuk ke kelas, dan teman-teman saya menyambut saya dengan antusias dan merayu saya untuk mentraktir mereka karena mereka tahu saya dapat banyak uang dari kemenangan itu. Piala yang saya bawa—mau disimpan di sekolah—dibawa bergilir di tangan teman-teman saya. Rasanya sulit menjaga agar kepala saya tidak bertambah besar.
Pelajaran pertama, sosiologi, saya ikuti dengan antusias dan pikiran yang segar. Ini tidak biasa karena menurut jadwal tidak resmi, saya seharusnya tidur di kelas sampai jam pelajaran ke tiga dimulai. Tapi itu bagus, toh ujian nasional sebentar lagi tiba, saya harus sedikit berbenah dalam mengatur segalanya.
Si guru sosiologi memberi sebuah tugas yang dikerjakan di perpustakaan. Tapi saya hampir tidak mengerjakan sama sekali karena saya sibuk memikirkan uang hasil lomba. Saya bingung, antara membeli sebuah DVDRW + RAM 1 Gb atau membeli sebuah VGA card yang mendukung directX 10. Kadang kepikiran saya harus dapat uang lagi untuk beli notebook tapi saya pikir saya belum perlu mobilitas seperti itu. Bel berbunyi dan saya belum menulis apapun. Akhirnya saya mengerjakan tugas itu secepat saya bisa. Setelah selesai saya berlari ke ruang guru untuk menyerahkan tugas yang terlambat itu.
Saat itulah saya hampir jantungan. Wakasek kesiswaan di sekolah saya menahan saya sambil tersenyum.
“Nanti, ambil uangnya di Bu Guru, ya..” katanya.
Saya jadi bingung, “Uang apa, Bu?”.
Saya dengar tidak begitu jelas, dia menyebut-nyebut tentang lomba. Sangka saya, saya diberi komisi lagi oleh sekolah karena menang olimpiade itu, tapi beliau menyebut-nyebut tentang penulisan. Otak saya berputar dan menemukan fakta bahwa saya pernah ikut sayembara penulisan cerpen yang diadakan Himpunan Mahasiswa Jurusan Komunikasi dan Penyiaran Islam Fakultas Dakwah IAIN Mataram. Saya ikut karena tertarik hadiah utamanya yang senilai Rp. 1.000.000. Tapi pengumuman sayembara itu terlambat 2 bulan sehingga hampir hilang sama sekali dalam ingatan saya.
Saya tanya saya juara berapa, wakasek menjawab tidak tahu. Dia suruh saya tanya seorang siswi yang namanya Nuansa Milyan (wakil OSIS) karena dia yang menitipkan hadiahnya ke wakasek dari acara pengumuman. Dia juga menjadi pemenang favorit bersama 2 siswi lain, mereka penulis hebat di sekolah saya.
Sayang, saya pemalu. Tapi karena ingin memastikan apakah saya mendapat juara atau tidak, saya berlari ke kelasnya, ke kantin, ke perpustakaan, ke UKS, ke lapangan basket, dan ke ruang guru tapi tidak menemukan mahluk itu. Saya jadi dag dig dug. Saya bilang sama wakasek kalau saya tidak menemukan anak itu. Akhirnya beliau menyuruh saya kembali ke kelas dan balik ke kantornya saat jam istirahat.
Saya tidak bisa tenang waktu itu. Setiap beberapa menit saya menanyakan kepada siapa saja, apakah bel istirahat sudah berbunyi. Saya mondar-mandir, teriak nggak karuan, dan mengusili teman-teman saya.
Ketika bel berbunyi saya berlari tergesa-gesa sampai tersandung-sandung dan menyenggol siswa lain. Sesampai di ruang wakasek, beliau sudah siap dengan amplop di tangannya, di sana tertulis ‘Rp. 1.000.000.’. Saya seperti mau pingsan. Ini pertama kalinya saya menghasilkan uang sebanyak itu tanpa bantuan orang lain. Saya menulis cerpen sendiri dengan ide sendiri, diketik di komputer sendiri dan diantar oleh saya sendiri ke panitia dengan mobil angkutan kota sepulang sekolah. Usaha saya sendiri!
Mulai saat itu saya yakin saya punya potensi di bidang penulisan. Karya saya diakui di atas 147 naskah lain dari seluruh NTB. Ini benar-benar luar biasa buat saya.
Cerpennya akan saya posting beberapa hari lagi berikut cerpen buatan peserta dari satu sekolah yang namanya saya sebutkan tadi.
Saya berniat menulis sebuah buku dan mudah-mudahan buku itu menjadi best seller!
