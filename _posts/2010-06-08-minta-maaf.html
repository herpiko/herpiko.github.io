saya minta maaf

kesalahan-kesalahan saya

berantakan, buah tangan saya sendiri

saya minta maaf

kelancangan saya

menginjak batas

saya minta maaf

kepura-puraan saya

yang saya kira seperti itulah wajah dunia

saya minta maaf

ketidaktahuan saya

alasan sebenarnya di balik semua ini

saya minta maaf

janji yang diingkari

karena takut

saya minta maaf

memiliki emosi seperti ini

tak terkendali

saya menyesal sekali
