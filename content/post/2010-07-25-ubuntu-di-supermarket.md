kemarin diajak pergi, mampir di supermarket di daerah cakranegara-sweta kota ruko antah berantah buat beli batere 9 volt (untuk UTP cable tester). terus saya kaget liat kasirnya.

itu desktop jaunty jackalope. terus kepala saya muter2 lihat deretan lain, 4 atau 5 komputer kasir, semuanya pakai ubuntu!

terus saya ngomong, "ubuntu... ubuntu...," sambil senyum-senyum ndak jelas di depan kasir. tapi saya kira wajah cewek itu seperti anggap saya orang gila, jadi saya memalingkan muka. saya lihat baterai 9 volt itu discan pakai code bar scanner. harga-harganya langsung muncul di layar ubuntu. saya baca apa aplikasi kasirnya, tapi saya lupa sekarang  jadi tidak saya tulis di sini. sialan keren dah!

besok-besok, saya mau datang ke sana, pura-pura beli pocary sambil bawa kamera (buat jepret ubuntu-nya), dan minta tanda tangan kasirnya mungkin. >.(o)

tumben sekali saya lihat penggunaan perangkat lunak terbuka di sini (mataram, dalam usaha/bisnis/pendidikan), palingan di leptop temen-temen tukang oprek. dan denger-denger PNS sekarang ndak bisa maen zuma lagi, karena harus pakai ubuntu dan ikuti pelatihannya. haha.
