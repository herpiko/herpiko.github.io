awal cerita, saya dikasih hape bekas cuma2 sama seseorang. saya kira itu motorala apa gitu, habis tombol2nya mirip sekali, taunya hape china. isinya, katrok ah. :(

tapi tak apa, toh dikasih, gratis pula, lumayan buat cadangan.

nah, saya coba cari2 fasilitas yang penting : platform java dan wap browser. fakta parah, java tidak ada. wap browser ada, tapi jadul. koneksi maksimal hape ini adalah gprs.

saat saya coba buka gmail, langsung muncul peringatan bahwa hanpon ini tidak mensupport gmail. silakan mengunduh versi java-nya. lha, java ndak ada kok.

saya coba2 buka yahoo, langsung bisa!

jadi?

solusinya adalah, mensinkron akun gmail dan yahoo.

gmail : forwarding ke email yahoo

yahoo : menjadikan gmail sebagai primary sender.

beres deh. jika ada email masuk di gmail maka akan masuk pula salinannya ke yahoo. kalau ngirim lewat yahoo, maka akan terkirim sebagai akun dari gmail. singkronlah. lumayan buat ngecek dan balas imel sewaktu2. :D

NB : btw, kenapa gmail tidak bisa diakses melalui wap browser ini? saya pernah mencoba membuka gmail di wap browser hp nokia monokrom (lupa serinya), pesan yang sama juga muncul. buka2 halaman2 web tertentu juga muncul peringatan 'is your browser support http/1.0 compliant? parah, padahal sudah pakai embel2 m.*. karena itulah saya mengambil kesimpulan : wap browser jadul. sok tau ah.

overall : hapenya jelek, tapi tetep syukur.
