sesekali tak apalah isinya windos, lagi pula disuruh blek.

kadang, di leptop2 keluaran baru, CD XP SP 2 sering mogok instal.

how to : keluarin CD XP SP2-nya, masukkin CD XP SP1. lanjutkan. balikin lagi CD XP SP2 setelah berhasil masuk tahap partisi. lanjut sampai selesai.
