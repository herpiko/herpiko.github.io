ini beneran, saya ndak boong.

saya beli adapter usb2ide portabel untuk hdd usang dari kakak saya, sebuah hdd laptop, PATA/IDE seagate 40GB 5400rpm. awalnya harddisk itu masih bisa digunakan, saya cek apakah ada badsector, dan syukurlah masih baik-baik saja. saya mengisinya dengan beberapa file yang saya kira penting untuk dibawa-bawa.

besoknya saya colokin lagi ke komputer dan harddisk itu tidak terdeteksi, cuma bunyi *'ngung... ngung...'*. saya jengkel sekali karena sudah buang uang untuk beli apadternya. karena jengkel yang tak bisa ditahan saya **membenturkan hdd itu ke tembok** lalu dicoba lagi, siapa tahu bisa. (jangan ditiru). tetap saja tidak bisa dan saya terus membenturkannya berkali2 dan kadang2 menepukkan ke tangan. akhirnya saya **membantingnya ke lantai**, keras. saya tinggal, tidak saya pungut.

malamnya, saya bersungut-sungut dan kakak saya nyeletuk, "itu mungkin kabelnya"

saya langsung kaget. aduh, iya ya siapa tau kabel usbnya yang bermasalah. ternyata memang benar kabel itu tertukar, kabel lama memang rusak sementara kabel baru masih ada di tas saya. saya nepuk jidat dan nyari2 harddisk tadi terus nyobain lagi pakai kabel yang baru.

langsung terdeteksi dan termount dan dibuka otomatis oleh nautilus. wuih! gak rusak!

tapi tunggu dulu, setelah diperlakukan semena-mena seperti itu, pastilah ada cacat, minimal badsector. saya langsung lakukan low level format. kalau macet berarti badsector. saya pakai [ini](http://hddguru.com/software/2006.04.12-HDD-Low-Level-Format-Tool/). lama memang, tapi lancar! gak rusak!

saya langsung masukin hdd-nya ke casing dan dibaut sambil ngelus-ngelus sayang. :mrgreen:

oh, tentu saja saya tidak akan membantingnya lagi.

NB : ini berbeda dengan harddisk biasa. mungkin di harddisk kecil laptop/harddisk portabel sudah ada antigetar atau apalahnamanya meski saya tidak membaca ada keterangan demikian. hal yang sama tidaklah terjadi di harddisk desktop saya yang dulu. CPUnya roboh ke samping dan harddisk itu jadi superlamban dengan badsector di tiga tempat berbeda.
