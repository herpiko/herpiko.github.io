Olimpiade Teknik 2008 UNUD Bali : Web Design

Date: 2008/11/13
Categories: kehidupan
Tags: bali, fraxis, lomba, olimpiade, olimpiade teknik, web design

Pagi tadi benar-benar menegangkan. Kondisi : saya diutus sekolah ke bali (mengutuskan diri) sama teman saya (frxs). Ada acara olimpiade teknik 2008 yang diadakan Universitas Udayana Bali. Saya ikut mata lomba web design (sebenarnya nggak tahu apa-apa). alhasil 1 bulan sebelumnya saya tidak pernah belajar (pelajaran sekolah), pulang sekolah langsung gigit komputer ngerjain web (isinya amburadul).

Kami (saya dan teman saya serta seorang guru dari sekolah) berangkat ke Bali dari Mataram, NTB, Naik kapal ferry yang lambat. Habis tu nungguin rombongan dari Selong (Lombok Timur) yang katanya mau ikut juga. Dari teman saya, dia bilang peserta web design ada 18 orang, otomatis saya jadi pesimis. Karya saya hanya biasa-biasa saja. Saya menaruh harapan pada menu flash drop down yang beda dengan peserta lain juga sedikit CSS dan Javascript.

Waktu sampai di pelabuhan Padangbai, Bali, kami dijemput mahasiswa Unud yang gagah2 pake satu mobil kijang dan satu bis mini karena tambahan 16 orang dari Selong. Rombongan selong terbagi dua sekolah, satu mengambil rancang bangun jembatan dan satu lagi mengambil karya tulis ilmiah. Hanya saya berdua bersama frxs yang berasal dari Mataram, Smanda.

Kami diantar ke hotel mutiara yang murah meriah (nggak ada pelayan yang mengantarkan barang saya lalu menunggu diberi tip) begitu juga rombongan Selong. Malam pertama di bali diisi dengan berjuang sama si frxs latihan dan menyusun web sampe jam 12 malam, saya sendiri nambah sampe jam dua karena sebelumya komputer yang saya bawa dipinjam frxs. Laptop frxs lagi kena virus ‘virut’ yang benar-benar ganas (menginfeksi setiap berkas ekstensi .exe di setiap partisi) jadi mau nggak mau komputer itu harus diinstall ulang.

Paginya kami dijemput lagi dari hotel Mutiara ke kampus Unud, ternyata jauh sekali, juga membosankan. Sudah capek nunggu di mobil, panas lagi. Sampai di kampus, banyak sekali siswa setingkat SMA/SMK yang lagi standby menunggu acara dimulai dan mengurus administrasi lomba. Pendek kata, kami ikut-ikutan sampai masuk ke acara pembukaan.

Akhirnya rombongan per mata lomba dipanggil dan lomba dimulai. Lomba dimulai dan kami diberi waktu 2 jam untuk menyelesaikan web kami. Saya tidak bohong kalau saya bilang tangan saya bergetar terus sampai lomba selesai, sering salah klik gara-gara gerakan mouse jadi kacau karena tangan saya gemetar. Saya menyelesaikan dengan terburu-buru dan sangat cepat. Sepertinya saya menarik perhatian mahasiswa panitia karena mengetikkan keyboard keras-keras dan cepat juga memukulkan mouse berkali-kali(maksudnya menggerakkan mouse dengan cepat). Saya lihat ada beberapa web peserta lain yang lebih baik dari saya tapi frxs bilang mungkin hanya kami berdua yang menyisipkan script lebih banyak ke web kami. Saya panik waktu web saya jadi kacau dibuka dengan firefox namun normal dengan IE. padahal waktu latihan semalam, yang terjadi adalah sebaliknya.

Saat saya panik, si frxs tenang-tenang saja. Mungkin dia ada sedikit masalah dengan script karena lebih banyak coding daripada menysun gambar dan artikel. Lomba selesai begitu juri menilai. Saya ragu dengan wawasan si juri. Mungkin di antara semua peserta, hanya kami berdua yang menggunakan tag marquee (tulisan bergerak ke samping). Ketika si juri (mungkin dosen muda) menilai kami berdua, ia agak heran dengan teks yang bergerak itu.

“Ini buat pake apa?”

“Marquee, pak”

“Marquee apa?”

“Tag marquee, pak. html.. script…” jawab saya kacau karena gugup.

“Apa itu?”

Karena tidak ingin menggurui, saya alihkan pembicaraan. Saya gerakkan pointer ke menu flash buatan saya dan dia bisa lihat dropdown yang aneh tapi beda, “Ini buat pake flash” kata saya dan dia mulai tanya hal lain. dia hanya periksa ukuran berkas keseluruhan web dan mencoba link-link, itupun tidak semuanya. Hanya itu saja, dan dia pergi ke peserta lain. Saya agak menyesal tidak menjelaskan dengan gamblang seperti frxs, supaya dia tahu kalau saya menggunakan beberapa script yang bagus dan susah payah saya hafal dan pelajari.

Tapi tak apa, saya harap mahasiswa panitia memeriksa lagi web-web kami dan menilai ulang.

Akankah sya menang? Saya tidak berani berharap terlalu banyak karena saya benar-benar masih bodoh. Tapi saya harus berharap karena sekolah sudah membiayai saya ikut olimpiade ke bali (berlibur). Saya seperi para pejabat yang sering melakukan perjalanan dinas (perjalanan wisata) yang sebenarnya tidak perlu (pake uang negara lho). Pemenang olimpiade ini diumumkan dua hari lagi, berarti tanggal 15 november. saya berusaha menjaga pikiran saya agar terus positif agar apa yang saya khayalkan (jadi pemenang) agak terealisasi.

Tulisan ini saya posting malam-malam sekali setelah shoping sama si frxs dan pak Joko (guru dan pembimbing kami). saya berburu cd linux (dapet 3 distro, redhat 64bit prossessor, centos5.2,dan nusantara3mahakam) di Ramayana, Denpasar. Saat ini diposting, frxs lagi tidur(kecapekan, dia beli buku yang bagus sekali tapi dibaca sambil tidur), juga pak Joko. saya sih, melek terus, gatelan lihat 3 cd linux di tangan saya, juga majalah linux dan buku-buku komputer yang kami beli di toko Gunung Agung. Saya ngenet pake 3G wirelessx pak Joko tapi pake kartu im3 saya, mungkin habis 1500 rupiah.

Intinya, saya sedang berharap saya menang di olimpiade yang jarang saya ikuti seperti ini.
