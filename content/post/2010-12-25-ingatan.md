1. saya sering lupa dimana meletakkan barang yang berukuran kecil, seperti ponsel, kunci, UFD, etc. saya bisa mengingat dengan baik dimana barang2 itu setelah saya duduk tenang lama sekali dan barangkali seharian.

2. saya lupa semua teman2 saya saat sma (baru dua tahun) kecuali sedikit orang yang masih sering saya temui, tetapi saya ingat semua nama panggilan teman-teman sekolah dasar saya, 41 orang tidak kurang tidak lebih.

3. saya lupa dan tidak pernah benar-benar ingat semua janji2 ketemuan kecuali jika saya menuliskannya di buku catatan atau dipasangkan ke pengingat seperti di ponsel. meskipun saya tidak pernah membuka buku catatan itu untuk melihat, saya tetap ingat karena saya menulisnya.

4. saya selalu lupa dengan semua yang pernah saya susun dan rencanakan. itu sebabnya saya selalu terlihat tidak konsisten dan tidak tepat janji. terutama dengan mahluk yang bernama jadwal.

5. saya tidak ingat nama dan wajah seseorang yang lama tidak berjumpa jadi saya sering pura2 mengenalnya, dan kalaupun saya mengingat namanya, saya sering tidak ingat siapa persisnya dan bagaimana saya bertemu pertama kali, jadi saya menunggunya berbicara tentang masa lalu sampai saya ingat.

6. hal-hal yang paling sering merusak ingatan adalah peristiwa-peristiwa yang menyita perhatian. misalnya saya sudah berjanji tentang hal ini dan itu bahkan saya sudah menuliskannya, kemudian saya berlibur di suatu tempat barang sehari maka saya lupa semua janji sampai orang atau pengingat ponsel mengingatkan.

7. kadang buku catatan sama sekali tidak berguna, karena : saya menuliskan dan kemudian saya lupa membawa buku catatan itu sendiri.

8. ada beberapa hal yang benar2 saya tidak bisa mengingatnya dan bahkan membantahnya meskipun orang lain sudah mengingatkan secara detil. saya merasa orang itu merekayasa dan orang itu mengira saya berpura-pura.

9. seperti semua orang, saya pasti ingat jika itu rutin atau berkesan sekali, baik itu indah maupun buruk.
